This is a review of different integration techniques, which you may find useful during the experiment.

Basic Integration Rules:
∫ k dx = kx + C
∫ x^n dx = (x^(n+1))/(n+1) + C (n ≠ -1)
∫ e^x dx = e^x + C
∫ a^x dx = (a^x)/(ln a) + C (a > 0, a ≠ 1)
∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
∫ sec^2 x dx = tan x + C
∫ csc^2 x dx = -cot x + C
∫ sec x tan x dx = sec x + C
∫ csc x cot x dx = -csc x + C
