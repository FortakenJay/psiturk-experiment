This is a review of different derivative rules, which you may find useful during the experiment.

Basic Derivative Rules:
d/dx (k) = 0
d/dx (x^n) = nx^(n-1)
d/dx (e^x) = e^x
d/dx (a^x) = a^x ln(a)
d/dx (sin x) = cos x
d/dx (cos x) = -sin x
d/dx (tan x) = sec^2 x
d/dx (cot x) = -csc^2 x
d/dx (sec x) = sec x tan x
d/dx (csc x) = -csc x cot x
